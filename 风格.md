# 海拓斯特（Hytorist）前端当前风格总结（已实现版）

## 1. 风格定位
- 品牌方向：机械设计品牌为主，科技感为辅。
- 视觉关键词：硬朗、工程感、目录化、可读性优先。
- 当前主风格：黑 / 白 / 橙三色体系（深底 + 白字 + 橙色强调）。

## 2. 品牌与语言规范
- 中文品牌名：海拓斯特。
- 英文品牌名：Hytorist。
- 导航语言规则（当前已修正）：
  - 中文站只显示中文导航与中文下拉项。
  - 英文站只显示英文导航与英文下拉项。
  - 不做中英混排。

## 3. 色彩系统（Tailwind Token）
- 深色基底（gunmetal）：
  - `#090a0d` / `#14161b` / `#1e2128`。
- 中性灰（steel）：
  - `#2d3139` ~ `#e7eaef`。
- 强调橙（signal）：
  - `#ff6a1a`（主强调）
  - `#ff7f33` / `#ffa46b`（层级强调）
- 正文与标题（ink）：
  - 标题白：`#ffffff`
  - 正文浅灰：`#f4f5f7` / `#d8dbe0`。

## 4. 字体系统
- 正文：`Manrope + Noto Sans SC`。
- 展示标题：`Rajdhani + Sora`（工程/机械感）。
- 编号与技术信息：`JetBrains Mono`。

## 5. 导航栏（V2 当前实现）
- 结构：深色固定顶栏 + 机械导轨式导航容器 + 橙色激活下划线。
- 桌面端：
  - 当前项白底黑字，普通项白字半透。
  - Hover 显示 Mega Menu（编号 + 单语标题 + 右箭头反馈）。
- 移动端：
  - 抽屉式菜单，样式与桌面一致。
  - 语言切换与“获取报价”按钮保留。

## 6. 首页风格（Home）
- Hero：轮播大图 + 深色遮罩 + 网格纹理 + 橙色进度线。
- 首屏布局：左侧品牌文案/指标，右侧 CAD 展示模块。
- CAD 区：支持模型加载与基础交互（旋转/缩放/复位），并与轮播主设备联动。
- 信息表达：
  - 指标卡（响应、项目、经验、设备数量）。
  - 工程标签带（法兰紧固、液压动力单元、定制服务等）。
  - “机械要点”编号块（M-01~M-04）。

## 7. 栏目页风格（Portal）
- 通用首屏：统一 Hero 头图、章节编号、面包屑。
- 列表页（新闻）：时间线与分隔线增强。
- 网格页（产品/案例）：统一卡片比例与机械边框。
- 文章页（关于/技术/服务）：统一段落间距与信息注释风格。
- 分页信息：按实际条目数计算并展示（避免固定写死）。

## 8. 组件与细节规范
- 圆角：整体偏小（机械硬边），常用 2px。
- 边框：1px 主边框，局部内描边/高光。
- 阴影：偏硬阴影，不使用大面积霓虹发光。
- 动效：低频、功能性动效为主（淡入、进度、轻扫描），避免炫光。

## 9. 按钮规范
- 主按钮（Primary）：橙底白字，用于关键 CTA。
- 次按钮（Secondary）：白底深字，Hover 转橙色语义。
- 导航与菜单反馈：Hover/Active 统一使用橙色作为状态识别。

## 10. CMS 与内容覆盖兼容
- 前端继续兼容 CMS 覆盖字段（首页与栏目页内容）。
- 字段缺失时回退默认内容。
- 对异常乱码文本做保护性回退，避免前台出现脏文本。

## 11. 当前可继续优化的方向（下一步）
- 导航栏再做“商务化压缩版”：高度、间距、字重进一步统一。
- 首页首屏信息密度再平衡：减少视觉噪音，提升第一屏转化路径。
- 实拍设备图与 CAD 模型素材统一编号，提升品牌一致性。
